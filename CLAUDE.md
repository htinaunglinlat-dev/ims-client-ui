@AGENTS.md
Create a modern responsive company website and blog platform for a tech company called “Lucky7andOne”.

The overall design should feel modern, clean, minimal, professional, and slightly futuristic.

Use smooth animations, rounded cards, soft shadows, glassmorphism elements, modern typography, and a clean grid layout.

Support both Dark Mode and Light Mode with a toggle switch in the navigation bar.

The website should be fully responsive for desktop, tablet, and mobile.

==================================================

GLOBAL DESIGN STYLE

==================================================

Design inspiration:

- Modern SaaS websites

- Startup portfolio websites

- Medium + Stripe + Vercel inspired layouts

- Clean developer-focused aesthetic

Color palette:

- Primary accent: blue / indigo gradient

- Clean white backgrounds in light mode

- Deep dark gray backgrounds in dark mode

- Use subtle gradients and glowing UI accents

Typography:

- Modern sans-serif typography

- Large bold hero headings

- Comfortable reading spacing for blogs

Add smooth hover animations and transitions.
